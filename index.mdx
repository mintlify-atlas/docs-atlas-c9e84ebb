  <Steps>
    <Step title="Clone the repository">
      Clone the portfolio source code to your local machine:
      ```bash
      git clone https://github.com/himanshubalani/portfolio_source.git
      cd portfolio_source
      ```
    </Step>
    
    <Step title="Install dependencies">
      Install Flutter dependencies using pub:
      ```bash
      flutter pub get
      ```
    </Step>
    
    <Step title="Customize your content">
      Edit the content in `lib/pages/` to add your own information, projects, and skills. Update colors in `lib/consts/style.dart` to match your brand.
    </Step>
    
    <Step title="Run the application">
      Start the development server:
      ```bash
      flutter run -d chrome
      ```
      Your portfolio will open in Chrome at `http://localhost:port`
    </Step>
  </Steps>
