## Get your portfolio running

Follow these steps to set up and customize your Flutter web portfolio. You'll have a working portfolio in less than 10 minutes.

<Steps>
  <Step title="Clone the repository">
    Clone the portfolio source code to your local machine:
    
    ```bash
    git clone https://github.com/himanshubalani/portfolio_source.git
    cd portfolio_source
    ```
    
    <Note>
    Make sure you have Flutter installed on your system. Visit [flutter.dev](https://flutter.dev/docs/get-started/install) if you need to install Flutter first.
    </Note>
  </Step>

  <Step title="Install dependencies">
    Install all required Flutter packages:
    
    ```bash
    flutter pub get
    ```
    
    This will install all dependencies listed in `pubspec.yaml`, including:
    - GetX for state management and routing
    - Google Fonts for typography
    - flutter_screenutil for responsive sizing
    - flutter_animate for animations
    - And more...

    <Accordion title="Verify Flutter installation">
    You can verify your Flutter setup with:
    ```bash
    flutter doctor
    ```
    This command checks your environment and displays a report of the status of your Flutter installation.
    </Accordion>
  </Step>

  <Step title="Customize your content">
    Before running the portfolio, customize it with your own information:

    ### Update personal information
    
    Open `lib/pages/homepage.dart` and update:
    ```dart
    Text(
      'Hello there, ',  // Change greeting
      // ...
    ),
    NeoBrutalContainer(
      text: '   Himanshu Balani ↵',  // Change to your name
      path: '/about',
      // ...
    ),
    ```

    ### Add your social links
    
    In the same file (around line 292), update the social links array:
    ```dart
    final socialLinks = [
      {
        'tooltip': 'Email',
        'icon': FontAwesomeIcons.envelope,
        'url': 'https://mail.google.com/mail/u/0/?to=your-email@example.com&fs=1&tf=cm',
        'color': AppColors.email
      },
      // Update other links...
    ];
    ```

    ### Customize colors
    
    Edit `lib/consts/style.dart` to change the color scheme:
    ```dart
    class AppColors {
      static const Color primaryPurple = Color(0xFF90a8ed);  // Your primary color
      static const Color lightBlue = Color(0xFFB4D6F8);      // Your accent color
      // Customize other colors...
    }
    ```

    <Tip>
    See the [Customization](/customization/colors) section for detailed guides on personalizing colors, fonts, and content.
    </Tip>
  </Step>

  <Step title="Run the application">
    Start the Flutter development server:
    
    ```bash
    flutter run -d chrome
    ```
    
    Your portfolio will open in Chrome. The development server supports hot reload, so any changes you make to the code will immediately appear in the browser.

    <Accordion title="Run on other devices">
    You can also run on other devices:
    ```bash
    # Run on macOS
    flutter run -d macos

    # Run on Windows
    flutter run -d windows

    # Run on connected mobile device
    flutter run

    # Run on specific device
    flutter devices  # List all devices
    flutter run -d <device-id>
    ```
    </Accordion>

    <Note>
    The first run might take a few minutes as Flutter builds the application. Subsequent runs will be much faster.
    </Note>
  </Step>
</Steps>

## Verify your setup

Once the application is running, you should see:

1. **Homepage** - Animated pill background with your name and motto
2. **Navigation bar** - Links to Home, Projects, About, and Socials
3. **Theme toggle** - Dark/light mode switcher in the app bar
4. **Social links** - Floating action buttons with your social profiles

<Tip>
Try pressing these keyboard shortcuts on the homepage:
- **D** - Opens Figma design
- **C** - Opens GitHub profile
- **Enter** - Navigates to About page
</Tip>

## Project structure

Here's what each directory contains:

```
portfolio_source/
├── lib/
│   ├── consts/           # AppColors, AppStyles, TooltipStyles
│   ├── pages/            # homepage.dart, about.dart, projects.dart, contact.dart
│   ├── presentation/     # App bars, hover effects, animations
│   ├── utils/            # Navigation utilities
│   ├── widgets/          # ProjectCard, NeoBrutalContainer, etc.
│   └── main.dart         # Application entry point
├── assets/
│   └── images/           # Project screenshots and assets
├── web/                  # Web configuration (index.html, manifest.json)
└── pubspec.yaml          # Dependencies and project metadata
```

## Common issues

<AccordionGroup>
  <Accordion title="Flutter not found">
    If you get a "flutter: command not found" error:
    1. Make sure Flutter is installed
    2. Add Flutter to your PATH
    3. Run `flutter doctor` to verify installation
    
    See [Flutter installation guide](https://flutter.dev/docs/get-started/install) for platform-specific instructions.
  </Accordion>

  <Accordion title="Package version conflicts">
    If you encounter package version conflicts:
    ```bash
    flutter pub upgrade
    flutter pub get
    ```
    
    This will resolve dependency conflicts and fetch compatible versions.
  </Accordion>

  <Accordion title="Hot reload not working">
    If hot reload isn't working:
    1. Press `r` in the terminal to manually hot reload
    2. Press `R` to hot restart
    3. Stop and restart the server with `flutter run -d chrome`
  </Accordion>

  <Accordion title="Chrome not launching">
    If Chrome doesn't launch automatically:
    1. Check that Chrome is installed
    2. Run `flutter devices` to see available devices
    3. Try running with `flutter run -d web-server` and open the URL manually
  </Accordion>
</AccordionGroup>
