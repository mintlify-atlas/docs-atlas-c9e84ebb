## Overview

The portfolio uses a comprehensive color system defined in `lib/consts/style.dart`. The `AppColors` class provides a centralized color palette with over 40 predefined colors organized into categories.

## Customizing Colors

### Change Primary Color

To change the main brand color, update `primaryPurple` in `lib/consts/style.dart:133`:

```dart
static const Color primaryPurple = Color(0xFF6B5CE7); // New purple
```

### Add New Color

Add a new color constant to the `AppColors` class:

```dart
class AppColors {
  // ... existing colors
  
  // Your custom colors
  static const Color customBlue = Color(0xFF3498db);
  static const Color customGreen = Color(0xFF2ecc71);
}
```

### Page Background Colors

Each page uses different background colors. Update these in the respective page files:

#### Homepage Background

In `lib/pages/homepage.dart:62`:

```dart
backgroundColor: isDark ? AppColors.black : AppColors.primaryPurple,
```

#### Projects Page Background

In `lib/pages/projects.dart:32`:

```dart
backgroundColor: isDark ? AppColors.black : AppColors.lightPeach,
```

#### About Page Background

In `lib/pages/about.dart:34`:

```dart
backgroundColor: isDark ? AppColors.black : AppColors.skyBlue,
```

## Animated Background Colors

The animated pill backgrounds can be customized by changing the `color` parameter:

```dart
AnimatedPillsBackground(
  rows: 10,
  columns: 24,
  color: AppColors.discord, // Change this color
  duration: Duration(seconds: 4),
)
```

<Note>
When changing colors, ensure sufficient contrast for accessibility. The neobrutalism design relies on high contrast between elements.
</Note>

## Theme-Aware Colors

The portfolio supports both light and dark themes. Use conditional color selection:

```dart
color: isDark ? AppColors.primaryPurple : AppColors.black
```

Where `isDark` is defined as:

```dart
bool get isDark {
  return Theme.of(context).brightness == Brightness.dark;
}
```

## Tooltip Colors

Tooltips use theme-aware colors defined in `TooltipStyles` at `lib/consts/style.dart:89`:

```dart
static BoxDecoration decoration({required bool isDark}) {
  return BoxDecoration(
    color: isDark ? AppColors.black : Colors.white,
    border: Border.all(
      color: isDark ? AppColors.brightOrange : AppColors.black,
      width: 2,
    ),
    borderRadius: BorderRadius.circular(8),
  );
}
```

<Tip>
Use the existing color palette as a starting point. The colors are carefully selected to work well together in the neobrutalism design style.
</Tip>

## Best Practices

1. **Consistency** - Use colors from the `AppColors` class rather than defining inline colors
2. **Contrast** - Maintain high contrast between foreground and background for readability
3. **Theme Support** - Always provide both light and dark theme color variants
4. **Semantic Naming** - Name colors based on their purpose or appearance, not their usage
5. **Border Colors** - In neobrutalism design, borders are typically black with 2-3px width
