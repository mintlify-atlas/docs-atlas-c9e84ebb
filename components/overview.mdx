## Key Features

All components in this library share common characteristics:

### Theme Support
Every component adapts to light and dark modes automatically:
- **Light Mode**: Colored backgrounds with black borders and shadows
- **Dark Mode**: Black backgrounds with neon-colored borders and glowing effects

### Responsive Design
Components use `flutter_screenutil` for consistent sizing across devices:
```dart
fontSize: MediaQuery.of(context).size.width <= 900 ? 12.w : 4.w
```

### Consistent Typography
All text uses the Outfit font family from Google Fonts:
```dart
fontFamily: GoogleFonts.outfit().fontFamily
```

### Neobrutalism Styling
Signature design elements include:
- Bold 2-3px borders
- Offset shadows (typically 3-6px)
- Rounded corners (12-20px radius)
- High contrast color combinations

## Utility Components

### OnHover Builder

The `OnHover` widget provides smooth hover animations:

```dart
OnHover(
  builder: (bool isHovered) {
    return YourWidget();
  },
)
```

**Location**: `lib/presentation/onhover.dart:1`

**Features**:
- Lifts element 10px on hover
- 200ms animation duration
- Smooth easing curve

### BoxCircles

Decorative circles used in headers:

```dart
BoxCircles(
  hoverEnabled: true,
  color: AppColors.lightPeach,
)
```

**Location**: `lib/widgets/neobrutalbox.dart:112`

## Usage Example

Here's how to combine multiple components:

```dart
import 'package:flutter/material.dart';
import 'package:portfolio/widgets/neobrutalbox.dart';
import 'package:portfolio/widgets/projectcard.dart';
import 'package:portfolio/consts/style.dart';

class MyScreen extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return Scaffold(
      body: Padding(
        padding: const EdgeInsets.all(16.0),
        child: NeoBrutalBox(
          headerText: 'My Projects',
          headerColor: AppColors.lightPeach,
          containerChild: ProjectCard(
            projectName: 'Flutter Portfolio',
            projectDescription: 'A modern portfolio with neobrutalism design',
            imagePath: 'assets/images/project.png',
            projectLink: 'https://github.com/username/repo',
            languages: ['Flutter', 'Dart', 'Firebase'],
            buttonText: 'View Project',
          ),
        ),
      ),
    );
  }
}
```
