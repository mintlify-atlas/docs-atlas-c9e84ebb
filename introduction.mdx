## Welcome to Portfolio

This Flutter web portfolio is a modern, responsive template designed to help developers showcase their work, skills, and experience with style. Built with Flutter and Dart, it features a distinctive **neobrutalism design** with bold colors, thick borders, and vibrant shadows that make your content stand out.

## What is this portfolio?

This is a production-ready Flutter web application that serves as a personal portfolio website. It's the same portfolio used by [Himanshu Balani](https://github.com/himanshubalani) and has been open-sourced for developers to use and customize.

<Note>
This portfolio is optimized for web deployment and works on all modern browsers. It's fully responsive and provides an excellent experience on both desktop and mobile devices.
</Note>

## Design Philosophy

### Neobrutalism

The portfolio embraces **neobrutalism**, a design trend characterized by:

- **Bold borders**: 2-3px black borders on all interactive elements
- **Vibrant shadows**: Offset shadows (3-5px) for depth without blur
- **High contrast**: Strong color combinations for visual impact
- **Geometric shapes**: Simple, clear shapes with minimal rounding
- **No gradients**: Flat colors for clarity and boldness

<Tip>
The neobrutalism style is intentionally bold and playful. It works especially well for creative portfolios where you want to make a strong visual impression.
</Tip>

### Responsive Design

The portfolio adapts seamlessly across devices:

- **Desktop (&gt;900px)**: Multi-column layouts, side-by-side content, full animations
- **Tablet (700-900px)**: Adapted layouts with reduced columns
- **Mobile (&lt;700px)**: Single-column stacked layouts, simplified animations

All components use `LayoutBuilder` and `flutter_screenutil` to provide optimal layouts at any screen size.
